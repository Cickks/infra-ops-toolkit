# Enterprise Homelab Documentation

This folder contains portfolio-ready evidence for the enterprise homelab.

Use these subfolders for hands-on documentation:

- `Architecture/` - master plan, system design, roadmap, network baseline, and design decisions.
- `Servers/` - server-level notes for `DC01`, `DHCP01`, `FILE01`, `PRINT01`, `LINUX01`, and future
  infrastructure hosts.
- `Clients/` - domain-joined workstation build notes, user validation, and help desk scenarios.
- `Network/` - IP addressing, DNS, DHCP, routing, firewall, packet capture, and topology notes.
- `Backups/` - backup, restore, VM export, snapshot, and rollback evidence.
- `SOPs/` - repeatable operating procedures for common administration tasks.

Evidence should include the objective, systems touched, steps performed, validation results,
problems encountered, fixes, rollback plan, lessons learned, and interview relevance.
