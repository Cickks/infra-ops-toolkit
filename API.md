# API

This repository does not currently expose an application API.

If a portfolio website, automation service, or AI Steve service is added later, document the API
here with:

- Purpose
- Authentication
- Endpoints
- Request and response examples
- Error handling
- Rate limits
- Logging and observability
- Security considerations

## Current Operational Interfaces

The current project interfaces are operational rather than application-level:

- Active Directory administration
- DNS and DHCP administration
- PowerShell Remoting
- SSH administration
- File and print services
- Linux administration
- Security and vulnerability scanning
- Documentation and evidence review
