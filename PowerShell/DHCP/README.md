# DHCP PowerShell

Use this folder for DHCP reporting and validation scripts.

Useful commands:

```powershell
Get-DhcpServerv4Scope
Get-DhcpServerv4ScopeStatistics
Get-DhcpServerv4Lease -ScopeId 192.168.50.0
Get-DhcpServerv4OptionValue -ScopeId 192.168.50.0
```

Document the server, scope, command, and validation result for each report.
