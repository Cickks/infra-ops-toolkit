# Linux

Purpose: Linux administration, SSH, Docker, Portainer, Raspberry Pi, Phase 11 evidence, and Phase 12 advanced Linux/self-hosting evidence.

Start with `Documentation/Server_Documentation/LINUX01/README.md` for active `LINUX01` commands.

Use subfolders:

- `Bash/` - shell scripts and command notes.
- `Backups/` - Linux backup and restore runbooks.
- `Cron/` - scheduled task notes and validation.
- `Documentation/` - Linux evidence and runbooks.
- `SSH/` - SSH setup and troubleshooting.
- `Docker/` - container practice.
- `NFS/` - Linux NFS share planning and evidence.
- `Portainer/` - container management UI notes.
- `ProductionReadiness/` - Phase 12.0A and 12.0B LINUX01 practice and INFRA01 production-readiness runbooks.
- `RaspberryPi/` - Raspberry Pi tooling and edge lab work.
- `Samba/` - Linux SMB share planning and evidence.
- `SelfHosting/` - Phase 12 service selection, app planning, and hosting decisions.
- `Systemd/` - custom service unit files, lifecycle notes, and troubleshooting.
